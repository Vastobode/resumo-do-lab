# resumo-do-lab
Este repositório contém o resumo das lições aprendidas durante o desenvolvimento do lab na FATEC

A apresentação basicamente mostra como a AWS/Azure funciona por trás das soluções que as empresas usam no dia a dia. Em vez de pensar cada conceito separado, tudo se mistura pra dar mais segurança, desempenho e praticidade. O A ideia é mostrar como a estrutura da nuvem e a divisão de responsabilidades entre a empresa e o provedor trabalham juntas pra fazer tudo funcionar sem dor de cabeça.
